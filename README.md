# 🚀 VRAPI JET

> **Simple. Fast. Lightweight.**

**VRAPI JET** adalah framework backend sederhana yang dibangun di atas **Node.js** dan **Express.js** untuk membantu developer membangun REST API dengan struktur proyek yang bersih, ringan, dan mudah dikembangkan.

Framework ini dirancang agar proses pengembangan menjadi lebih cepat tanpa konfigurasi yang rumit, sehingga cocok untuk proyek kecil hingga enterprise yang membutuhkan fondasi backend yang sederhana namun fleksibel.

---

## ✨ Features

* ⚡ Lightweight & Fast
* 📁 Clean Project Structure
* 🌐 REST API Ready
* 🔒 Middleware Support
* 🛠️ Easy Configuration
* 📦 Built with Express.js
* 🔧 Easy to Extend
* 🚀 Production Ready

---

## 📦 Requirements

* Node.js 18+
* npm atau yarn

---

## 🚀 Installation

Clone repository:

```bash
git clone https://github.com/vembririyan/VRAPI_JET.git
```

Masuk ke folder project:

```bash
cd vrapi-jet
```

Install dependencies:

```bash
npm install
```

Jalankan aplikasi:

```bash
npm run dev
```

atau

```bash
npm start
```

## 🎯 Philosophy

VRAPI JET mengusung prinsip:

* **Simple** — Mudah dipelajari dan digunakan.
* **Minimal** — Hanya menyediakan fondasi yang dibutuhkan.
* **Flexible** — Bebas dikembangkan sesuai kebutuhan proyek.
* **Maintainable** — Struktur kode yang rapi dan mudah dipelihara.

---

## ❤️ Why VRAPI JET?

Banyak framework memiliki konfigurasi yang kompleks. VRAPI JET hadir sebagai alternatif yang lebih ringan sehingga developer dapat lebih fokus pada pengembangan fitur daripada konfigurasi proyek.
